# LiqueDT on iPhone

LiqueDT is prepared as an installable, mobile-responsive Progressive Web App (PWA). It includes iPhone safe-area spacing, touch-sized controls, a standalone display mode, offline shell caching, and Apple home-screen artwork.

## Install on an iPhone

1. Publish the `LiqueDT-App` web files to an HTTPS address.
2. Open that address in Safari.
3. Tap **Share**.
4. Tap **Add to Home Screen**, then **Add**.

It will open from the home screen like an app. Live charts, calendar, and news still require internet access and the server-side feed gateway. An `.exe` is Windows-only and cannot be installed on iPhone.
